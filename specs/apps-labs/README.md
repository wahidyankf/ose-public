# Experimental App Specs

Gherkin acceptance specifications for experimental applications in `apps-labs/` — framework
evaluations, tech stack comparisons, and proof-of-concept implementations.

## When to Use This Directory

Use `specs/apps-labs/` when the application is:

- A tech stack evaluation or POC not yet committed to production
- A framework comparison with no production deployment planned
- An experimental implementation where the spec may evolve rapidly

Use `specs/apps/` when the application is production-bound or has a stable spec.

## Graduation

Experimental specs may graduate to `specs/apps/` once the implementation is promoted from
`apps-labs/` to `apps/`.

## Current Contents

This directory is currently empty — all previous experimental specs have graduated.
